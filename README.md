# MirAI School of Technology – AI Builder Track Internship 2026

This repository contains all assignments completed during the **Virtual Summer Internship 2026 (AI Builder Track)** conducted by **MirAI School of Technology**.

The internship focused on building practical AI-powered applications using Python, Streamlit, prompt engineering, state management, and user-centric interface design.

---

## Assignments Overview

### Assignment 1 – The Identity Echo Interface

A Streamlit-based application designed to collect and display user information through an interactive interface.

#### Key Features

* Structured user data collection
* Dynamic form inputs
* Interactive Streamlit UI
* Input validation and display logic
* Clean and responsive layout

#### Skills Learned

* Streamlit fundamentals
* Form handling
* User interaction design
* Data collection workflows

---

### Assignment 2 – Multiverse Chatbot

An AI-powered chatbot capable of responding with different personalities and communication styles.

#### Key Features

* Multiple chatbot personalities
* Dynamic prompt customization
* Real-time AI responses
* Interactive sidebar controls
* User-friendly chat interface

#### Skills Learned

* Prompt engineering
* API integration
* Conversational AI development
* Streamlit application design

---

### Assignment 3 – Memory Vault (Stateful Chatbot Upgrade)

An enhanced version of the Multiverse Chatbot that introduces persistent memory using Streamlit session state management.

#### Key Features

* Multi-turn conversation support
* Persistent chat history
* Session-based memory storage
* History preservation during UI updates
* Improved chatbot experience

#### Skills Learned

* State management
* Session handling
* Streamlit architecture
* Stateful application development

---

### Assignment 4 – AI Image Studio Upgrade

An AI-powered image generation application allowing users to create images using different artistic styles and creative settings.

#### Key Features

* AI image generation
* Multiple art style options
* Custom generation settings
* Interactive Streamlit dashboard
* User-controlled creativity options

#### Skills Learned

* Generative AI integration
* Image generation workflows
* API-based AI applications
* User experience enhancement
